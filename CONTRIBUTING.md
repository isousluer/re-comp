# Contributing to Re-Comp

Thank you for your interest in contributing! Here's how to get started.

## Getting Started

```bash
git clone https://github.com/isousluer/re-comp.git
cd re-comp
npm install
npm run dev
```

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Commit: `git commit -m "feat: your feature description"`
5. Push: `git push origin feature/your-feature`
6. Open a Pull Request

## Guidelines

- Keep changes focused and minimal
- Follow the existing code style (Vanilla JS, no frameworks)
- Test in Chrome, Firefox, and Safari before submitting
- All processing must remain 100% client-side — no server calls

## Reporting Issues

Open an issue on GitHub with a clear description and steps to reproduce.

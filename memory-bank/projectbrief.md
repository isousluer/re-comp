# Re-Comp - Project Brief

## Proje Özeti
Re-Comp, blog görselleri için yerel bir web uygulamasıdır. Görselleri belirtilen çözünürlüğe yeniden boyutlandırır ve belirlenen kalite oranında sıkıştırır.

## Temel Gereksinimler
- Görsel yükleme (drag & drop + dosya seçici)
- Belirtilen çözünürlüğe yeniden boyutlandırma (en-boy oranı korunarak)
- Kalite parametresi ile sıkıştırma (JPEG/WebP) ve Quantization (PNG)
- Tamamen yerel çalışma — sunucuya yükleme yok
- Premium, modern UI tasarımı

## Hedef Kullanıcı
Blog yazarları — özellikle usluer.net blogu için öne çıkarılan görsellerin optimize edilmesi.

## Kapsam
İlk sürüm: Tek görsel yükleme → boyutlandırma → sıkıştırma → indirme.
